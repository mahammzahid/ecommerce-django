# 🛒 Django Ecommerce Web Application

A full-featured **Ecommerce Web Application** built using **Django**, implementing real-world shopping functionality including cart management, checkout system, order handling, and stock control.

---

## 🚀 Features

* 🛍️ Product Listing (Card-based UI)
* 🔍 Search Products by Name
* 🛒 Add to Cart System
* ➕ Update Quantity / Remove Items
* 📦 Checkout with Address, Phone & Payment Method
* 🧾 Order Creation & History
* 📊 Stock Management (Auto-update after order)
* 📄 Invoice Page for Orders
* 🔐 User Authentication (Login Required)

---

## 🧱 Tech Stack

* **Backend:** Python, Django
* **Frontend:** HTML, CSS, Bootstrap
* **Database:** SQLite
* **Authentication:** Django Session Auth

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git remote add origin https://github.com/mahammzahid/ecommerce-django.git
cd ecommerce-django
```

### 2. Create virtual environment

```bash
python -m venv env
env\Scripts\activate   # Windows
```

### 3. Install dependencies

```bash
pip install -r requirements.txt
```

### 4. Run migrations

```bash
python manage.py makemigrations
python manage.py migrate
```

### 5. Create superuser

```bash
python manage.py createsuperuser
```

### 6. Run server

```bash
python manage.py runserver
```
